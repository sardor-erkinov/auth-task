# Application for Government

**Technology**: Next.js

[Live Demo](https://zulfiya.vercel.app/)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Sardo0r/ariza.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install 
  or 
  yarn install
```

Start the server

```bash
  npm run start 
  or
  yarn start
```
